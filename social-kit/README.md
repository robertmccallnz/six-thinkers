# Six Thinkers — Social Media Kit

**Free social media kit for [The Kiwi Dialectic's Six Thinkers series](https://robertmccallnz.github.io/six-thinkers/).**
Live preview: **https://robertmccallnz.github.io/six-thinkers/social-kit/**

## What's in this folder

| Path | What it is |
|---|---|
| `index.html` | Public preview page — visual gallery + copy-to-clipboard scripts across five platforms. |
| `scripts.md` | The full script document as portable Markdown (X, Substack Notes, Facebook, Bluesky, LinkedIn). |
| `visuals/` | 21 PNGs, 1080×1080 square, each in the site's exact palette (`#0a0a0a` bg, `#f4ecd8` cream, Bebas Neue + Inter, per-thinker accent colours). |

## Visuals index (21 files)

- `04_master_toolkit.png` — the master card, mirrors the site H1
- `06_series_index.png` — all six courses in one shareable grid
- `05_koha_support.png` — the koha ask
- `01_thinker_*.png` — six thinker portrait cards
- `02_quote_*.png` — six pull-quote cards
- `03_prompt_*.png` — six discussion-prompt cards
